# Services

## PlatformService

PlatformService is responsible for managing platform-related data and operations.

- `GET /api/platforms`: Retrieves all platforms.
- `GET /api/platforms/{id}`: Retrieves a specific platform by ID.
- `POST /api/platforms`: Creates a new platform.

## CommandService

CommandService is responsible for handling command-related data and operations.

- `GET /api/commands`: Retrieves all commands.
- `GET /api/commands/{id}`: Retrieves a specific command by ID.
- `POST /api/commands`: Creates a new command.
